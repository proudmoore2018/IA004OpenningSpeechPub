# 整理的要点

1.依据演讲稿、切分组块；

2.修正速记稿的错误；

3.增补 keynote 中的图片在恰当位置；

4.将 keynote 中的文字为主的图片，将文字提取出来，放在正文中；

5.对开智的黑话（一些术语）进行解释或增加案例说明（可以从我的博客检索）;假设读者是第一次看开智的公众号，并不理解开智的黑话。

6.最后，使用写匠修正错别字，润色文字。写匠是开智发布的人工智能辅助写作软件：http://www.aiwriter.cn
